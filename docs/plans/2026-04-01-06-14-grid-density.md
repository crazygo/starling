# Grid Density Reduction Plan (50% sparser)

## Background
用户反馈当前“地平网格（方位/高度）”和“天球网格（赤经/赤纬）”过于密集，希望整体稀疏 50%。

## Goal
将两类网格线数量减半，同时保持现有绘制逻辑与视觉风格不变。

## Implementation
1. 定位网格绘制代码：`lib/widgets/star_chart.dart` 中 `_drawHorizontalGrid` 与 `_drawEquatorialGrid`。
2. 将网格主间隔从 `30°` 调整为 `60°`：
   - 地平网格：`altitude` 与 `azimuth` 循环步进 `30 -> 60`
   - 天球网格：`declination` 与 `right ascension` 循环步进 `30 -> 60`
3. 不修改曲线采样精度（每 `2°` 采样）以避免线条锯齿变化。

## Validation
1. 运行静态检查/测试，确认无编译与分析错误。
2. 人工确认设置页面打开网格开关后，线条数量约为原先 50%。

## Risks & Notes
- 网格更稀疏后，部分用户可能觉得定位参考减少；如后续需要可再增加“网格密度”可配置项。
