# 星星渲染条件设置合并计划

## 背景
用户反馈当前“视觉分组”设置不符合产品预期：
- “显示非属于星座/星官的星星”不应是独立开关。
- 该项应与“背景星星阈值”合并为单一设置“星星渲染条件”。
- UI 不要 group 风格（RadioGroup），改为“单个设置项 + 右侧弹出候选列表”。

## 目标
1. 将两项设置合并为一个枚举：
   - 星星可见度 小
   - 星星可见度 中
   - 星星可见度 大
   - 仅参与星座连线的星星
2. 设置页改成单行条目，右侧弹出列表选择。
3. 渲染逻辑与新枚举保持一致。
4. 跑 analyze/test 验证。

## 实施
1. `SettingsService`
   - 删除 `showNonConstellationStars`。
   - 新增 `StarRenderCondition` 枚举及持久化。
2. `SettingsPage`
   - 删除旧开关与 radio 组。
   - 新增“星星渲染条件”条目 + 右侧下拉菜单。
3. `ExplorePage` / `StarChart`
   - 参数改为传递 `starRenderCondition`。
4. `_StarPainter`
   - 依据条件决定：
     - 小/中/大：分别映射标签阈值与非成员星可见策略。
     - 仅参与星座连线：只绘制参与连线星（非成员不绘制、不参与竞争标签）。

## 风险
- 设置 key 迁移会导致旧布尔值失效（可接受，使用新默认值兜底）。
